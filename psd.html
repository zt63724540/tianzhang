53.Maximum Subarray: if the cum sum below zero, then re-start the cum sum link

88.Merge Sorted Array: the key of this question is the last of array 1 has available room to make use of, so start to fill in the end of array 1 by comparing the end of array 1 and array 2 link

118.Pascal’s Triangle: first create a list of list with value 1 and then every element equals to two elements of the previous level link

122.Best Time to Buy and Sell Stock II: cumulative all the increment change link

  • Math

7.Reverse Integer: use [::-1] and be careful about the negative sign link

9.Palindrome Number: use [::-1] to detect link

168.Excel Sheet Column Title： make use of ord and dividend is 26 link
